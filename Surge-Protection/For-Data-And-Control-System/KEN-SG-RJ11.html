KEN-SG/TC Series Surge Protector for RJ45 & RJ11

Requirement class to VDE 0675:D
Requirement class to EN61643-11: Type 3
Requirement class to IEC61643-1: Class III

Application Range:

KEN-SG/RJ11 series surge protective devices for signal are applied to signal protection of telephone dialing equipment, which can prevent the system or device from permanent damage or transient interruption arising from inductive overvoltage, over current and the other transient surge voltage caused by surge or industrial noise, etc. They are widely used to be against surge in electronic communication system device, such as a telephone, a fax, a program control machine room, etc.

Main Features:

1. Aluminum profile shell, attractive and durable, good shielding effect.
2. Standard RJ11 interface, convenient to use ,reliable connection.
3. Adopt multilayer protection circuit, selecting the latest high-speed surge protection device of quick response, low output residual voltage and super transmission performance.
